Detailed Description

their are 81 phonemes in the SAM.
they are numberd by 0-80.
every phoneme has property: id(0-80) name(1-2characters), normal-length, stressed-length, freq123, amp123 consonant-flags

phoneme name has 1-2 characters. the name is stored in phonemeNameTable1 (for first char) and phonemeNameTable2 (for second char) .
phonemeLengthTable is the default length of the phoneme.
phonemeStressedLengthTable is the default stressed length of the phoneme.
the unit of the length is frame. one frame is approxmitly 10ms.
the phoneme have three types.

  1. vowel, which is generated by 3-formants. wave=f1wav*f1amp + f2wav*f2amp * f3wav*f3amp
  2. unvoiced-consonant. which is just recorded and played when need.the data is stored in: wave= consonantSampleTable .
  3. voiced-consonant. which is wave=1+2. wave=f1wav*f1amp + f2wav*f2amp * f3wav*f3amp + consonantSampleTable

freq1data freq2data freq3data and ampl1data ampl2data ampl3data is the phoneme formant freq and amp.
sampledConsonantFlags is the phoneme flag, include the consonantSample pos and consonantSample length.
