◆ RenderOnce()

void RenderOnce ( )

RenderOnce RENDER THE PHONEMES IN THE LIST.

The phoneme list is converted into sound through the steps:

  1. Copy each phoneme <length> number of times into the frames list, where each frame represents 10 milliseconds of sound.
  2. Determine the transitions lengths between phonemes, and linearly interpolate the values across the frames.
  3. Offset the pitches by the fundamental frequency.
  4. Render the each frame.

input: phonemeIndexOutput phonemeLengthOutput stressOutput

output: buffer bufferpos
